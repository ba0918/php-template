# PHP project boilerplate
[![Build Status](https://travis-ci.org/ba0918/php-template.svg?branch=master)](https://travis-ci.org/ba0918/php-template)

## Requirement
- PHP7.0, PHP7.1
- [Composer](https://getcomposer.org/)

## Development
### Setup

```
composer install
```

## Test
```
composer run test
```
